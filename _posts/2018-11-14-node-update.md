#### node 업데이트 하는 방법

NPM 캐시 지우기   
```
sudo npm cache clean -f
```
```
sudo npm install -g n
```

최신 Node.js 버전 설치   
```
sudo n stable
```

또는 특정 버전을 선택하고 다음과 같이 설치   
```
sudo n 0.8.20
```
