####node 업데이트 하는 방법
<br><br>
NPM 캐시 지우고 재설치
```
sudo npm cache clean -f
sudo npm install -g n
```
<br><br>
최신 Node.js 버전 설치   
```
sudo n stable
```
<br><br>
또는 특정 버전을 선택하고 다음과 같이 설치   
```
sudo n 0.8.20
```
